# Box Breathing

Box breathing is a simple calming technique used to steady the nervous system
and bring attention to the present moment. It is called "box" because each of
the four steps takes an equal count, like the four sides of a square.

## How to practice

1. **Inhale** slowly through your nose for a count of 4.
2. **Hold** your breath gently for a count of 4.
3. **Exhale** slowly through your mouth for a count of 4.
4. **Hold** empty for a count of 4.

Repeat the cycle four to six times, or for about one to two minutes. Keep the
counts comfortable — if 4 feels strained, shorten it; the equal sides matter
more than the exact number.

## When it helps

Box breathing is useful before a stressful conversation, when winding down for
sleep, or anytime you notice your mind racing. It is a wellness practice, not a
medical treatment; if you have a breathing or heart condition, check with a
clinician before using paced-breathing techniques.
